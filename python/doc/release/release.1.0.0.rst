Py-M.E.S.S.: 1.0.0 Release Notes
================================

.. contents::

1.0.0 is the initial release.


Release highlights:
-------------------

    * Solver for sparse dense algebraic Lyapunov and Riccati equations.
    * Interface to |blas| Level 3 Lyapunov and Stein equation solvers.
    * Solver for positive dense algebraic Riccati equations.
    * Sparse-Dense Sylvester Equation solvers.
    * Support for First Order, Second Order, Index-1 DAE and Index-2 DAE Systems.

None.
